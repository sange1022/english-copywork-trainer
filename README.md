# 中英文打字临摹

在线版：https://sange1022.github.io/english-copywork-trainer/

## 直接使用

双击 `scripts/启动英文临摹.command`，浏览器会自动打开。

首次使用前，电脑需要安装 Node.js 20 或更高版本。

## 怎么练习

1. 粘贴中文或英文内容，网页会自动识别模式。
2. 点击“开始练习”。
3. 英文模式只需输入英文字母；忽略大小写，空格和标点自动跳过，并显示中文翻译。
4. 中文模式使用中文输入法逐字输入汉字；空格、标点、数字和英文字母自动跳过，不显示翻译。
5. 输入错误时字符会变红，按退格删除错误后继续。
6. 完成一句约 1 秒后自动进入下一句。
7. 英文模式下，点击原文里的英文单词可查看中文释义。

翻译和单词释义需要联网。免费翻译服务偶尔不可用，但不会影响打字练习。
如果当前句显示“翻译暂不可用”，可以点击旁边的“重新翻译”直接重试。

关闭启动时出现的终端窗口，即可停止网页服务。
