# 英文打字临摹

## 直接使用

双击 `scripts/启动英文临摹.command`，浏览器会自动打开。

首次使用前，电脑需要安装 Node.js 20 或更高版本。

## 怎么练习

1. 粘贴英文内容。
2. 点击“开始练习”。
3. 对照英文和中文翻译，只需输入淡色句子中的英文字母。
4. 大小写不作区分，空格和标点会自动跳过，不需要输入。
5. 输入错误时字符会变红，按退格删除错误后继续。
6. 完成一句约 1 秒后自动进入下一句。
7. 点击原文里的英文单词可查看中文释义。

翻译和单词释义需要联网。免费翻译服务偶尔不可用，但不会影响打字练习。
如果当前句显示“翻译暂不可用”，可以点击旁边的“重新翻译”直接重试。

关闭启动时出现的终端窗口，即可停止网页服务。
